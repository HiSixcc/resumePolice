# resumePolice（简历警察）
如果你是一个程序员，你想要找一份新的工作，简历准备是不可或缺的。

那么，
如何让自己的简历变得更加专业、更加具有深度；
如何排查简历中的问题、如何做进一步优化；
如何直击hr、面试官的灵魂；
如何精准化的针对简历和岗位jd做精准备战预案？

项目提供了精炼且经过一些真实实验数据验证过的提示词，以及dify的workflow关键性文件。
